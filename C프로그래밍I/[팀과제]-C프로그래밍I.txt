[팀과제] C언어 관련 질문 및 답변

1.
Q.
switch문에서 각 case마다 break;가 없다면 어떻게 되는가?
A.
case의 조건에 없는 값이라면 아무 것도 실행되지 않으며, case의 조건에 있는 값이라면 그 case부터 순차적으로 실행됩니다.

2.
Q.
int main()과 void main()의 차이점은 무엇인가요?
A.
main()을 종료할 때의 리턴값은 운영체제가 받게 되는데, 리턴값이 0이라면 정상적으로 종료된 것이고, 그 외의 값이라면 비정상적으로 종료된 것으로 판단합니다. 그러나 이것은 운영체제의 입장에서 중요할 뿐, 사용자의 입장에서는 그리 중요하지 않기 때문에 사용자에게는 차이점이 없습니다.

{

}

{

prn(we);

#include<stdio.h>
struct user
{
int age;
char n[80], t[80];
};
void prn(struct user* we)
{
printf("나이 이름 전화번호\n");
printf("%d %s %s\n",we->age,we->n,we->t);
we++;
printf("%d %s %s\n",we->age,we->n,we->t);
we++;
printf("%d %s %s\n",we->age,we->n,we->t);
we++;
}
void main()
{
struct user we[3]={{18,"jcshim","010-1111-2222"},{16,"jeshim","010-2111-2222"},{14,"jbshim","010-3111-2222"}};
prn(we);
}
